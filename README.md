# dungeon_master

Консольное приложение, на языке Ruby, которое которое имулирует простую dungeon RPG игру

На текущий момент, есть возможность менять комнату, просматривать статус Героя и вступать в битвы с монстрами:

#### Для старта игры
**bundle install***
**bundle install***
**ruby main.rb***
